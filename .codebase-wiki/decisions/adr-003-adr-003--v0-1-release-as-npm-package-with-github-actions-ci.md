# ADR-003: ADR-003: v0.1 release as npm package with GitHub Actions CI

> **Status**: Accepted

## Context
Phase 7 requires npm publishing, CI, and a public repo. The project has 239 tests passing, 7 examples, and all exit criteria met.

## Decision
Publish @kapy/script v0.1.0 on npm. Set up GitHub Actions CI for test, typecheck, and lint. Release as MIT license. Tag v0.1.0. Known limitations documented in CHANGELOG.

## Consequences
- (to be determined)

## Alternatives Considered
Could publish without CI, or wait for more polish. Decided to ship v0.1 now and iterate based on feedback.
