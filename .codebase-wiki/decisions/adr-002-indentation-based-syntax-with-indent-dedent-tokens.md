# ADR-002: Indentation-based syntax with INDENT/DEDENT tokens

> **Status**: Accepted

## Context
Need block structure similar to Python. Options: brace-based, significant whitespace with INDENT/DEDENT tokens, or offside rule.

## Decision
Use Python-style INDENT/DEDENT tokens generated by the lexer. 2-space indentation enforced. Tabs rejected at lex time. Only one indent level at a time (no skipping from 0 to 4 spaces). This eliminates brace-mismatch errors — a top category of AI-generated syntax errors.

## Alternatives Considered
Braces like TypeScript, offside rule, indentation-meaningful but brace-optional
